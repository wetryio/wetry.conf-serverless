# Google DNS

Malheureusement l'outil XIP.IO que nous allons utilisé durant ce Workshop n'est pas disponible sur les serveurs DNS belges.
Nous allons donc devoir utiliser celui de google.

* [Windows](#Windows)
* [OSX](#OSX)
